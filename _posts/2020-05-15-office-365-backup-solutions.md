First of all I would like to say that this blog is a personal opinion. It is not sponsored or biased in anyway.  
  
We all know we need an Office 365 Backup Solution since Microsoft doesn’t really offer one. But which Backup Solution do you need and why?   
  
I have compared 5 Office 365 backup solutions for 17 different criteria. This information is acquired by searching the vendors websites and contacting them about their products. Pricing is not covered in this article since this depends on volumes of users and whether you need to use your own storage, server etc…

### Conclusion:

My personal favorites are AvePoint and NetApp. AvePoint and NetApp have the richest featuresets and have the possibility to choose your own storage if you want to. Furthermore AvePoint and NetApp tick all the boxes for the criteria that I find important. These solutions can backup the most Office 365 products, are SaaS solutions and have a MSP Dashboard. The only downside is that they don’t support MFA natively. You can use Office 365 to log on to the portal and set MFA there. Additionally AvePoint has the option to set an Access List for the portal based on trusted IP’s.  
  
If you need an Office 365 Backup for a single organization and don’t need multitenancy my choice would be Barracuda. It has all the features you need but lacks the MSP features. I know I wouldn’t talk about pricing but if you have more features than you need, chances are that you are paying to much. So my choice therefore is Barracuda if you have a single tenant which you would like to backup.  
  
When you are an MSP and would like to view all your customers in a single solution you have more options. Veeam, Altaro and NetApp are options I think you should look at. 

If you want the richest featureset choose NetApp or AvePoint. If you would like to host the complete backup suite yourself choose Veeam. Veeam requires you to host the backup server and storage. Altaro has 2 downsides, it doesn’t support Microsoft Teams and you can’t select the storage location, the default is Europe. If you don’t mind these downsides Altaro is a possible solution for you. (It is also the cheapest)
