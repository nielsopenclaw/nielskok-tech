## About NielsKok.Tech

Hi, I'm Niels Kok — an IT professional specialising in End User Computing (EUC), Microsoft 365, and Modern Workplace solutions.

This blog covers hands-on guides, tips, and insights on:

- **Microsoft Intune** — device management, compliance policies, app deployment
- **Azure Virtual Desktop (AVD)** — session host deployment, image management, automation
- **Windows 11** — deployment, updates, configuration
- **PowerShell** — automation scripts and tooling
- **DevOps & Infrastructure as Code** — Bicep, Packer, Azure DevOps pipelines

Posts date back to 2020 and are migrated from the original WordPress site.
